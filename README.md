# Курсовая работа

Тема: Настройка и администрирование системы управления версиями и средами разработки

## Описание
Данный репозиторий используется в рамках курсовой работы для демонстрации
настройки системы управления версиями, контейнеризированной среды разработки
и базового конвейера CI/CD.

В качестве тестового приложения используется минимальный сервис,
предназначенный исключительно для проверки работоспособности инфраструктуры.
Разработка прикладного программного обеспечения не является целью работы.

## Структура проекта
- app/ — тестовое приложение
- tests/ — минимальные тесты
- Dockerfile, docker-compose.yml — контейнеризация
- .github/workflows — CI/CD
