# Курсовая работа

Тема: Настройка и администрирование системы управления версиями и средами разработки

Репозиторий используется для хранения конфигураций и демонстрации
настройки контейнеризированной среды и CI/CD.

## Структура проекта
- app/ — тестовое приложение
- tests/ — минимальные тесты
- Dockerfile, docker-compose — контейнеризация
- .github/workflows — CI
